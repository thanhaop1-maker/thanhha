### 🚀 Cách tôi "Build" chiếc Portfolio này bằng AI (Behind the Scenes)

Không cần phải bắt đầu từ con số 0, đây là quy trình 3 bước tôi đã áp dụng để tối ưu hóa thời gian xây dựng website cá nhân:

**Bước 1: Lên ý tưởng & Tạo Prompt thiết kế (Sử dụng ChatGPT / Gemini)**
Tôi đã dùng prompt sau để AI định hình phong cách và tạo ra bộ khung thiết kế:
> *"Bạn là một chuyên gia UI/UX. Tạo cho tôi một portfolio đơn giản bao gồm html + css. Giao diện dark hoặc white. Có một thay đổi thêm ở đầu trang. Thông tin cá nhân. Hà Trần Phú Thành, ngành Đồ hoạ kỹ thuật số. Công trình đã công bố: 12 năm phổ thông top 10, 1 bằng tốt nghiệp, thích đá bóng. Hãy cho tôi một bộ prompt có thể thiết kế tạo ra website như vậy"*

**Bước 2: Khởi tạo môi trường lập trình**
- Tạo tài khoản GitHub cá nhân.
- Tạo repository mới, thêm file `README.md` này để làm profile.
- Khởi chạy **GitHub Codespace** để có ngay một môi trường code trực tuyến tiện lợi mà không cần cài đặt phức tạp trên máy.

**Bước 3: Code giao diện cùng Claude AI**
- Sau khi có hướng dẫn và mockup từ Bước 1, tôi sử dụng **Claude AI** để trực tiếp viết mã nguồn HTML và CSS.
- Copy code thả vào Codespace, tinh chỉnh lại hình ảnh (như việc ghép ảnh đại diện và ảnh chú Husky) để hoàn thiện một trang web mang đậm dấu ấn cá nhân.
